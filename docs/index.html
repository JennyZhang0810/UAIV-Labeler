UAIV-Labeler

Build low-altitude remote-sensing datasets beyond boxes. UAIV-Labeler combines detection, oriented boxes, polygons, event QA, OCR/VLM review, metadata indexing, model pre-annotation, and benchmark export in one open-source workflow.

Supported Tasks

UAIV-Labeler is built for multi-task aerial observation data construction, not only object detection.

Detection Horizontal bounding boxes for common aerial-view objects and events.
Oriented Detection Rotated boxes for aerial-view objects with arbitrary direction.
Segmentation Polygon and mask-oriented review for area-level annotations.
Event QA Question review for urban governance and multimodal reasoning.
OCR / VLM Inspect OCR, VLM, and model answers next to image evidence.
Restoration Pair Pair and inspect degraded/restored low-altitude images for restoration tasks.

Why UAIV-Labeler?

CVAT and Label Studio are excellent general tools. UAIV-Labeler focuses on low-altitude remote-sensing dataset production where metadata, event QA, model-assisted review, and benchmark export are native parts of the same workflow.

One workbench for annotation, model review, and QA.

Annotators can switch between boxes, oriented boxes, polygons, and QA review. Dataset builders can organize image sources, model backends, metadata, and export formats without gluing multiple tools together.

Server-folder import indexes large image collections without copying original data.
Readable JSON remains the source of truth, with SQLite side indexes for filtering and status summaries.
Sample data is bundled, so a new user can clone the repository and run a local demo immediately.

Dataset Production Flow

A complete loop from raw imagery and metadata to model assistance, human review, quality control, benchmark export, and dataset release assets.

01 Raw Imagery Server folders, browser uploads, flight batches, and paired image assets.
02 Metadata Indexing Altitude, GPS, weather, scene, source, task type, timestamp, and review status.
03 Pre-annotation YOLO, SegEarth, SAM-style prompts, OCR/VLM backends, and custom HTTP predictors.
04 Human Review Detection boxes, oriented boxes, polygons, masks, QA answers, and paired-image checks.
05 Event QA Question options, model answers, ground truth, image evidence, and reasoning traces.
06 Quality Control Low-confidence flags, missing metadata checks, status summaries, and conflict review.
07 Benchmark Export COCO, VOC, YOLO, YOLO-OBB, DOTA, GeoJSON, Mask PNG, and QA JSONL.
08 Dataset Release Dataset Card, README assets, release notes, and paper-ready workflow summaries.

Roadmap

The current release already covers core review and export workflows. Next milestones focus on stronger segmentation, larger imagery, collaboration, and dataset release operations.

Available Detection and OBB Horizontal boxes, oriented boxes, polygons, QA review, and benchmark export are available in the current workflow.
Available Model pre-annotation YOLO, SegEarth, OCR/VLM, SAM-style services, and custom HTTP backends can be connected.
Next Interactive segmentation Click and box prompt segmentation with stronger SAM-style review interactions.
Next Large image tiling Tile-based browsing for large mosaics, wide-area imagery, and GeoTIFF-like data.
Next Multi-user collaboration Reviewer assignment, audit trails, team status tracking, and database-backed workflows.
Next Dataset release manager Dataset Card generation, export summaries, release checklists, and paper-ready assets.
Planned Full offline package Self-contained deployments for intranet, no-internet, and secure lab environments.
Planned Task template library Reusable templates for urban governance, ecology, agriculture, emergency response, and restoration tasks.
