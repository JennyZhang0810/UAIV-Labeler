# Sample Data

This folder provides a tiny runnable sample for the UAIV-Labeler.

Contents:

- `images/`: four low-altitude remote-sensing sample images for import and manual annotation trials.
- `qa/sample_qa.tsv`: a small QA review file paired with the images.
- `sample_metadata.json`: example Metadata schema for documentation and conversion scripts.

Try it in the platform:

1. Open the Workbench.
2. Choose Server Folder.
3. Browse or enter `sample_data/images`.
4. Click import.
5. Open QA Review and load `sample_data/qa/sample_qa.tsv`.
