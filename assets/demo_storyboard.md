# 15-second Demo GIF Storyboard

Use this storyboard to record the short GitHub demo GIF before release.

1. `0-3s`: Open the platform home page and click **Open Workspace**.
2. `3-6s`: In **Server Folder**, enter `sample_data/images` and import the sample images.
3. `6-9s`: Select one image; show the Metadata panel with altitude, GPS, weather, scene, and task tags.
4. `9-12s`: Click **Draw Box**, draw a bbox, and set the label to `vehicle` or `construction_machine`.
5. `12-15s`: Press `Ctrl+S`; show the saved toast and automatic move to the next image.

Recommended filename after recording:

```text
assets/demo.gif
```

Recommended tools:

- macOS: Kap or Screen Studio.
- Windows: ScreenToGif.
- Linux/server desktop: Peek or ffmpeg screen capture.
