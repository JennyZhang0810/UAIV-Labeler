# Changelog

## 0.1.0

Initial public release preparation.

- Metadata-first image indexing and filtering.
- Server-folder import without copying large image collections.
- Local upload for small trial batches.
- Manual bbox annotation with undo, clear all, next/previous image, and `Ctrl+S` save-next.
- Task modes for urban foundation model, ecological foundation model, and image restoration.
- Model registry with local, remote, mock, and custom HTTP backend options.
- QA TSV review and XLSX result-review utilities.
- Export support for JSON, COCO, VOC, and QA JSONL.
- Annotation history snapshots and task state machine.
- Spatial filtering and lead-time statistics.
- Docker quick-start support.
- Public sample dataset under `sample_data/`.
- README banner, workflow diagram, and demo GIF storyboard.
